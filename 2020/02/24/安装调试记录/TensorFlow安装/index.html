安装TensorFlow记录

  1. 安装anaconda

    安装过程中如果出现failed to create anacoda menu，先忽略让其安装完成，然后在安装的根目录输入如下命令即可。

    C:\Program Files\Anaconda3>python .\Lib_nsis.py mkmenus

  2. 卸载之前版本

    TensorFlow2.0改了语法，与斯坦福课程教的1.x不同，于是卸载。下次安装这种大型框架时要在版本号上多做一些功课。

  1. 将pip切换回国内源

    pip国内的一些镜像

      阿里云 http://mirrors.aliyun.com/pypi/simple/
      中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/
      豆瓣(douban) http://pypi.douban.com/simple/
      清华大学 https://pypi.tuna.tsinghua.edu.cn/simple/
      中国科学技术大学 http://pypi.mirrors.ustc.edu.cn/simple/

    修改源方法：

    临时使用：
    可以在使用pip的时候在后面加上-i参数，指定pip源
    eg: pip install scrapy -i https://pypi.tuna.tsinghua.edu.cn/simple –trusted-host pypi.tuna.tsinghua.edu.cn

    永久修改：
    linux:
    修改 ~/.pip/pip.conf (没有就创建一个)， 内容如下：

    [global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple

windows:
直接在user目录中创建一个pip目录，如：C:\Users\xx\pip，新建文件pip.ini，内容如下

   [global]
   index-url = https://pypi.tuna.tsinghua.edu.cn/simple

  1. 安装tensorflow

    打开 Anaconda prompt

    conda create -n tensorflow pip python=3.5

    pip install –upgrade –ignore-installed tensorflow_gpu==1.8.0 -i http://pypi.douban.com/simple –trusted-host pypi.douban.com

卸载tensorflow：

activate tensorflow

pip uninstall tensorflow-gpu

  1. 使用

    激活环境：activate tensorflow

    关闭环境：deactivate

结论：

他妈的不要在windows上装tensorflow坑的要死，新装一个linux或者虚拟机然后在里边装tensorflow耗时都比这短。

最后win还是用不成在ubuntu上装的，ubuntu就一个conda就完事儿了。最后发现最近版的cs224n已经不用tensorflow改用pytorch了，淦。
