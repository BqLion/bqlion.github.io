windows环境下使用hexo搭建hiker博客记录

搭建步骤

  • 购买域名

  • github创建个人仓库

  • 安装node.js

  • 安装hexo

  • 推送网站

  • 域名解析

  • 更换主题

  • 更改配置

  • 发布文章

  • 使用图床

1.域名购买

godaddy.com: 域名很全但价格较高，首年年费均价￥60左右,往后每年续费￥120左右。

阿里云：价格适中但个性化后缀较少,cc后缀域名无法备案

namesilo.com: 域名较全,价格最低,bqlab.cc十年￥280,最终在此购买。

2.github创建仓库

仓库名命名为bqlion.github.io，亲测改成其他域名网站不能正常访问，

3.安装node.js

下载地址：https://nodejs.org/en/download/

安装完成后检测是否成功：node -v

检测npm是否一起安装成功: npm - v

4.安装hexo

创建名为bqlion.github.io的文件夹，在这里打开cmd命令行

输入

npm install -g hexo-cli

初始化博客

hexo init bqlion.github.io

检查网站雏形

cd bqlion.github.io

hexo new test_my_site

hexo g

hexo s

打开localhost:4000 检查网站是否已经拉起

常用hexo命令

npm install hexo -g #安装Hexo
npm update hexo -g #升级 
hexo init #初始化博客

命令简写
hexo n  "我的博客" == hexo new  "我的博客"   #新建文章
hexo g == hexo generate   #生成
hexo s == hexo server   #启动服务预览
hexo d == hexo deploy   #部署

hexo server     #Hexo会监视文件变动并自动更新，无须重启服务器
hexo server -s    #静态模式
hexo server -p 5000     #更改端口
hexo server -i 192.168.1.1     #自定义 
IPhexo clean     #清除缓存，若是网页正常情况下可以忽略这条命令

5.推送网站

根目录下的_config.yml是站点配置文件，themes文件夹里的_config.yml文件是主题配置文件

将hexo与github关联起来：

打开_config.yml，修改最后几行为：

deploy:
type: git
repo: https://github.com/yuxuanart/yuxuanart.github.io
branch: master

其实就是给 hexo d这个命令做配置，让hexo知道要把blog部署在哪个位置

安装git部署插件

npm install hexo-deployer-git --save

分别输入三条命令（清理，生成，部署）

hexo clean 
hexo g 
hexo d

到这里博客就上线了

6.域名解析

登录https://www.dnspod.cn/

阿里云在添加两条yuxuan.art中添加两条记录

第一条记录值是yuxuanart.github.io，第二条记录值是yuxuanart.github.io的ip

ip地址获取方法是打开cmd

ping yuxuanart.github.io

点开github之前创建的仓库，点settings，设置custom domain,输入bqlab.cc

在namesilo.com中的Domain manager中将域名DNS服务器修改为：

ns1.alidns.com

ns2.alidns.com

在blog根目录/source文件夹里创建记事本，内容就是自己的网址

yuxuan.art

退出保存为CNAME,注意保存为所有文件而不是txt文件

然后输入

hexo clean
hexo g
hexo d

fatal : yuxuanart deny了bqlion的登入

git config --global user.name yuxuanart
git config --global user.email yuxuanart@126.com

7.更换主题

我使用的hiker主题，参照https://github.com/fluid-dev/hexo-theme-fluid设置

_config.yml修改红框主题为fliud

9.发布文章

可直接把.md文件拖进_post目录，注意在每个.md文件头部插入yml块，输入如下内容配置好标签和分类

categories:
- 算法
- CS229
tags:
- 机器学习
- 梯度下降

10.使用图床

  1. 注册七牛云，实名认证，新建存储空间
  2. 使用MPic图床神器，配置AK SK后即可使用，直接把图片拖拽上去即可在剪切板生成md插入图片链接
  3. 使用snipaste截屏工具，比windows自带工具好用

11.常用命令

hexo clean

hexo g

hexo d

hexo new <title>
